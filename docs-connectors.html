Connector Reference

Complete documentation for all 40+ DIME connectors. Select a category below to explore available connectors.

Industrial Protocol Connectors

Connect directly to PLCs, robots, and industrial equipment using native protocols.

OPC-UA

Source & Sink

Industry-standard protocol for industrial automation. Supports secure authentication, browsing, and subscription.

Key Features:

  • Secure client/server communication
  • Node browsing and discovery
  • Subscription-based monitoring
  • Certificate-based authentication

Example Configuration:

- name: opcua_client
  connector: OpcUa
  endpoint: opc.tcp://192.168.1.100:4840
  username: admin
  password: password
  items:
    - name: temperature
      node_id: ns=2;s=Temperature
    - name: pressure
      node_id: ns=2;s=Pressure

Modbus TCP

Source

Widely-used protocol for industrial devices. Read holding registers, input registers, coils, and discrete inputs.

Key Features:

  • TCP/IP and RTU support
  • Multiple data types
  • Batch reading optimization
  • Automatic reconnection

Example Configuration:

- name: modbus_plc
  connector: ModbusTCP
  address: 192.168.1.50
  port: 502
  slave_id: 1
  items:
    - name: temperature
      address: 40001
      type: float
    - name: status
      address: 10001
      type: bool

Ethernet/IP

Source

Connect to Allen-Bradley PLCs including ControlLogix, CompactLogix, and MicroLogix families.

Key Features:

  • Native tag-based addressing
  • Array and UDT support
  • Multiple PLC types
  • High-speed data collection

Supported PLCs:

  • ControlLogix
  • CompactLogix
  • MicroLogix
  • PLC-5
  • SLC-500

Siemens S7

Source

Direct connectivity to Siemens S7-300, S7-400, S7-1200, and S7-1500 PLCs.

Key Features:

  • Native S7 protocol
  • DB block access
  • Multiple data types
  • Optimized block transfer

MTConnect Agent

Source & Sink

MTConnect standard for manufacturing equipment. Embedded agent or external connectivity.

Key Features:

  • MTConnect 1.4+ support
  • Embedded agent capability
  • Custom data items
  • Real-time streaming

Fanuc Robot

Source

Direct connection to Fanuc robots using SNPX protocol for R-J3iB, R-30iA, and R-30iB controllers.

Available Data:

  • Joint positions
  • Cartesian coordinates
  • I/O status
  • Alarms and programs

Enterprise System Connectors

Integrate with databases, message queues, and enterprise applications.

MQTT

Source & Sink

Lightweight messaging protocol for IoT. Publish and subscribe to topics with QoS support.

Key Features:

  • MQTT 3.1.1 and 5.0
  • TLS/SSL encryption
  • QoS 0, 1, 2 support
  • Retained messages
  • Last will and testament

PostgreSQL

Sink

Write time-series and event data to PostgreSQL databases with automatic table creation.

Key Features:

  • Automatic schema creation
  • Batch inserts
  • TimescaleDB support
  • Connection pooling

Redis

Source & Sink

High-performance in-memory data store. Use for caching, pub/sub, and real-time analytics.

Supported Operations:

  • Key-value storage
  • Pub/sub messaging
  • Time-series data
  • Sorted sets for rankings

MongoDB

Sink

Document-oriented NoSQL database. Store complex hierarchical data with automatic indexing.

Key Features:

  • Automatic collection creation
  • Time-series collections
  • Aggregation pipelines
  • Change streams

ActiveMQ

Source

Enterprise message broker supporting JMS. Subscribe to queues and topics.

Key Features:

  • JMS 1.1 compliance
  • Topic and queue support
  • Message selectors
  • Durable subscriptions

HTTP Client/Server

Source & Sink

RESTful API integration. Create endpoints or consume external APIs.

Capabilities:

  • REST API server
  • Webhook receiver
  • HTTP POST/GET client
  • Custom headers

Cloud Service Connectors

Connect to cloud platforms and SaaS applications for hybrid deployments.

InfluxDB

Sink

Time-series database optimized for IoT data. Write metrics with high throughput.

Key Features:

  • Line protocol support
  • Automatic tagging
  • Batch writes
  • InfluxDB 2.0 and 3.0

Splunk HEC

Sink

Send events and metrics to Splunk via HTTP Event Collector.

Key Features:

  • Event and metric endpoints
  • Source typing
  • Index routing
  • Batch optimization

SparkplugB

Source & Sink

MQTT-based specification for industrial IoT. Birth certificates and data modeling.

Key Features:

  • SparkplugB 3.0
  • Birth/death certificates
  • Metric aliasing
  • Edge node modeling

Specialized Connectors

Industry-specific and specialized protocol support.

Haas SHDR

Source

Connect to Haas CNC machines for real-time monitoring and data collection.

Available Data:

  • Machine status
  • Program execution
  • Spindle load
  • Alarms and messages

ASC CPC

Source

Connect to ASC autoclave systems for process monitoring.

Process Data:

  • Temperature profiles
  • Pressure readings
  • Cycle status
  • Alarm conditions

Script

Source

Generate data using Lua or Python scripts. Ideal for testing and simulation.

Use Cases:

  • Test data generation
  • Complex calculations
  • Data simulation
  • Custom protocols

Common Connector Properties

All DIME connectors share these common configuration options:

Property Type Default Description
name string "Unnamed" Unique connector name
enabled boolean true Enable/disable connector
scan_interval integer 1000 Polling interval in milliseconds
items array [] List of data items to collect
